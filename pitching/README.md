# ssmpitch

Static web pitch deck for the SSM R&D Grant proposal:
AI-Based Prototype System for Early Warning of Corporate Non-Compliance.

Production URL: https://ssmpitch.vercel.app/

## Files

- `index.html` - slide content and layout
- `styles.css` - presentation styling
- `script.js` - slide navigation controls

## Local preview

```bash
python3 -m http.server 8001 --bind 127.0.0.1
```

Then open `http://localhost:8001/`.
